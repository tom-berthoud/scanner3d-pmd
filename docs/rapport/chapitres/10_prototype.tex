\subsection{Fonctionnalités déjà démontrées}

En mode simulation (\emph{mock} matériel) :

\begin{itemize}
  \item Cycle de scan complet sur 200 pas ($1{,}8^\circ$ par pas)
        avec objets synthétiques (sphère, cylindre, cube, formes
        libres).
  \item Extraction de ligne laser par seuillage $G-R$ + centroïde
        sous-pixel.
  \item Reconstruction par fusion des profils et filtrage
        d'\emph{outliers} KDTree.
  \item Export STL binaire et OBJ via \texttt{trimesh}.
  \item Interface web avec suivi temps réel des étapes (SSE),
        téléchargement du résultat et viewer 3D intégré.
  \item Couverture de tests unitaires sur la machine d'états et les
        étages de traitement.
\end{itemize}

Reste à démontrer sur matériel réel : acquisition caméra Pi,
pilotage moteur NEMA\,17 via DM320T, commutation laser par MOSFET,
cycle de bout en bout.

\subsection{Points ouverts}

\begin{itemize}
  \item Définition du banc de mesure de précision : objet de
        référence imprimé (sphère ou pièce étalon), protocole de
        comparaison nuage-à-nuage (distance de Hausdorff ou
        équivalent).
  \item Essai d'ergonomie avec utilisateurs tiers ($n=10$)
        conformément à \textbf{E7}.
  \item Durcissement logiciel sur matériel : gestion des
        \emph{timeouts}, récupération d'erreurs GPIO, reconnexion
        caméra.
  \item Photos et rendus du prototype final (à intégrer dans cette
        section pour le rapport final).
\end{itemize}
